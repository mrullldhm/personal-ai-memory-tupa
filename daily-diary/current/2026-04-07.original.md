### Local Run Workflow (for reference)

```bash
# Step 1 — .env
cd /home/tidea/projects/pdfgenerator
echo "PORT=3003\nAUTH_TOKEN=local-dev-token" > .env

# Step 2 — run
docker compose up --build

# Step 3 — smoke test
curl -X POST http://localhost:3003/pdf \
  -H "Authorization: Bearer local-dev-token" \
  -H "Content-Type: application/json" \
  -d '{"html": "<h1>Test</h1>"}' \
  --output /tmp/test.pdf

# Step 4 — point hermes at it (hermes/.env)
PDF_URL=http://localhost:3003/pdf
PDF_API_KEY=local-dev-token
```
